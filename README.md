# xfy-node-getstarted
从Node.js客户端调用科大讯飞API.

> 仅支持Linux64
Read ```https://www.npmjs.com/package/xfy-node```.

## 准备工作
```
cd resources
sudo unzip xfyun_Linux_voice_1135_5864ae2d.zip -d /opt/xfy-sdk
```

## 资源文件
**data.vioces**

*.silk 文件 - N/A
*.wav 文件 - 16k, 单通道
*.flac 文件 - 22k, 单通道

## 测试
```
cd xfy-node-getstarted
npm install
node index.js
```
